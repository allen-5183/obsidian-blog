## TabCopy使用簡介

1. 一次複製所有分頁的標題與網址。
    `「 TabCopy 」` 的使用方法很簡單，在  `Chrome` 安裝這款套件後，**點擊右上方的「 TabCopy 」按鈕一下**，就可以把當前分頁的**標題**與**網址**複製下來。

    **如果點擊「 TabCopy 」按鈕兩下**，可以一鍵複製目前視窗裡「所有分頁」的標題與網址，這真的省下許多來回複製貼上的步驟。

2.  設定複製格式：**純文字**、 **HTML**、 **自訂**  
    當然， `「 TabCopy 」` 不只可以複製標題與網址，他還能自訂我們想要複製的格式，例如複製成可以直接張貼的 `HTML` 語法，廣泛的支援讓 `「 TabCopy 」` 可以快速在論壇、 BBS、 網頁或任何你的編輯工具中轉貼文章。

## 安裝完成後

安裝網址：

```yaml
Chrome 線上應用程式商店
→ Tab Copy
```
安裝後，瀏覽器右上角會出現 `Tab Copy` 圖示。

# 功能說明

點擊工具列圖示後會看到：

|按鈕|功能|
|---|---|
|This tab|複製目前分頁|
|This window's tabs|複製目前視窗所有分頁|
|All tabs|複製所有視窗的所有分頁|
|All tabs by window|依視窗分組複製所有分頁|

## This tab

複製目前瀏覽頁面

例如目前開啟：

```
https://obsidian.md
```

按：

```
This tab
```

即可複製到剪貼簿。

## This window's tabs

複製目前 Chrome 視窗內所有分頁。

例如：

```
GoogleYouTubeObsidianGitHub
```

一次全部複製。

## All tabs

複製所有 Chrome 視窗中的全部分頁。

適合：
- 工作中的所有頁面備份
- 關機前保存工作環境
- 建立研究資料清單

## All tabs by window

依照視窗分組輸出。

例如：

```
工作視窗
├ GitHub
├ Jira
└ Confluence

學習視窗
├ Obsidian
├ YouTube
└ ChatGPT
```

適合：
- 工作紀錄
- 每日工作日誌
- Obsidian 知識管理

# Copy 1x as

截圖右下角：

```
Copy 1x as:URL
```

可以選擇輸出格式。

## URL

輸出：

```
https://obsidian.md
```

---

## Title

輸出：

```
Obsidian
```

---

## Markdown

最常用
輸出：

```
[Obsidian](https://obsidian.md)
```

貼到 Obsidian 立即變成可點擊連結。

## HTML

輸出：

```
<a href="https://obsidian.md">Obsidian</a>
```

適合網頁開發。

## Wiki

部分版本支援：

```
[[Obsidian]]
```

適合 Obsidian Wiki Link。


# Obsidian 最佳搭配

將格式改成：

```
Markdown
```

之後：

```
All tabs
```

複製結果：

```
- [Obsidian](https://obsidian.md)- [ChatGPT](https://chatgpt.com)- [GitHub](https://github.com)- [YouTube](https://youtube.com)
```

直接貼進 Obsidian：

```
## 今日研究資料- [Obsidian](https://obsidian.md)- [ChatGPT](https://chatgpt.com)- [GitHub](https://github.com)
```

即可形成研究紀錄。
---

✅ Show tab counts

```
顯示分頁數量
```

# 建議快捷鍵

畫面下方有：

```
Edit keyboard shortcuts...
```

可以設定：

例如：

```
Ctrl + Shift + C
```

複製目前分頁

或：

```
Ctrl + Shift + A
```

複製所有分頁

之後不用點圖示即可快速匯出網址清單。

## Roam-Highlighter

- 按〔Alt+X〕進入 `Roam-Highlighter` 模式
- 選取要複製的文字後按〔`Ctrl+X`〕
- 繼續複製本網頁裡的別的文字並按〔`Ctrl+X`〕
- 結束後到Obsidian按〔`Ctrl+V`〕
