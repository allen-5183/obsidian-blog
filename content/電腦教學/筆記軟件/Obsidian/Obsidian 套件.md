
## Image Toolkit

社群外掛：

```
Image Toolkit
```

功能：

- 點擊圖片放大
- 滾輪縮放
- 拖曳移動
- 全螢幕檢視
- 多張圖片切換

安裝：

```
設定→ Community Plugins→ Browse→ Image Toolkit
```

安裝後：

```
![[diagram.png]]
```

點擊圖片即可放大。

效果類似：

```
網站 LightboxGoogle PhotosNotion 圖片預覽
```


## Codeblock Customizer

安裝：

```
Codeblock Customizer
```

功能：

- Copy Button
- 顯示程式語言
- 行號
- 語法高亮
- 程式碼摺疊

例如：

````
```sqlSELECT *FROM users;```
````

顯示：

```
SELECT *FROM users;
```

右上角會出現：

```
SQL | Copy
```

一鍵複製。
