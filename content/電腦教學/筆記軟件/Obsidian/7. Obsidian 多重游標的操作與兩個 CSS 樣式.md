
## 多種游標操作

在編輯器中，使用多重游標以快速執行群體操作。

按 `Alt`/`option` 後(滑鼠游標變成加號) 再到特定位置按 `Click`，會出現長條游標，再到別的位置按 `Click`，會再出現第二個長條游標，此即多重游標，點擊結尾後輸入要插入的字串，如 `-` 或 `[[`，在長條游標處就會出現插入字串 . 在文件其他位置 `Click` 或按 `Esc`，即可脫離多重游標狀態。

**範例:** 

到一個網站 [[源自德國的ZK筆記整理術-打造第二大腦-RaNdy Lien-Medium]](https://randylien.medium.com/%E6%BA%90%E8%87%AA%E5%BE%B7%E5%9C%8B%E7%9A%84-zk-%E7%AD%86%E8%A8%98%E6%95%B4%E7%90%86%E8%A1%93-%E6%89%93%E9%80%A0%E7%AC%AC%E4%BA%8C%E5%A4%A7%E8%85%A6-35f204d612f4)上，複製帶有項目編號內容時，在新文件上貼上時，項目編號或符號不會自動帶上!

>有些情況下：

```
Alt + Click
```

會被 `Windows` 或顯卡驅動攔截。

輸入：
```
Alt + Shift + Click
```
##### 更可靠的方法

在 `Obsidian` 裡直接執行：

```
Ctrl + Shift + P
```

輸入：

```
Add cursor above
```

或：

```
Add cursor below
```

預設快捷鍵通常是：

```
Alt + Shift + ↑
Alt + Shift + ↓
```

## 項目符號線條樣式

讓**項目符號** 出現垂直線，容易識別為第幾層的關係。

`obsidian.css`

```css
/*=== Bullet Point Relationship Lines ===*/
.cm-hmd-list-indent, ul ul { 
  position: relative; 
}
.cm-hmd-list-indent::before, ul ul::before {
  content:'';
  border-left: 1px solid rgba(0, 122, 255, 0.55);
  position: absolute;
}
.cm-hmd-list-indent::before {
  left: 0; 
  top: -5px; 
  bottom: -4px; 
}
ul ul::before {
  left: -11px;
  top: 0;
  bottom: 0; 
```

## 隱藏 YAML 標示

`YAML` 雖不是 `Markdown`  語言的標準，但是有非常多的 `Markdown` 語法有支援在 `Markdown` 行開頭可以附上 `YAML` 的格式，這 `YAML` 的格式，如果在被內嵌到 `Obsidian` 筆記時，會破壞頁面顯示的樣子，那就可以將那些被內嵌的 `YAML` 將其隱藏起來。

`obsidian-hide-embded-YAML.css`

```css
/*=== Remove Yaml header from embeds ===*/
/* Remove embed yaml first separator */
.markdown-embed-content > hr:first-child { display: none; }
/* Remove embed yaml content */
.markdown-embed-content > hr:first-child + h2 { display: none; }
/* Remove embed yaml second separator (if empty) */
.markdown-embed-content > hr:first-child + hr { display: none; }
/* Remove embed yaml second separator */
.markdown-embed-content > hr:first-child + h2 + hr { display: none; }
```
