# 簡介 `SQL Server`

## 前言

1. 關於 `SQL Server`
   `SQL Server` 是 `Microsoft` 公司推出的關係型資料庫管理系統。具有使用方便可伸縮性好與相關軟體集成程度高等優點，也是一個全面的資料庫平台，使用集成的 **商業智慧型 (BI)** 工具提供了企業級的數據管理。
   ![](https://pic.allen5183.synology.me/MSSQL_BACKUP_166.png#w90)
   `Microsoft SQL Server` 資料庫引擎 (`SSDS`) 為關係型數據和結構化數據提供了更安全可靠的存儲功能，使您可以構建和管理用於業務的高可用和高性能的數據應用程式。

   版本名稱|版本說明
   --|--
   **企業版 `Enterprise`**|存取關鍵任務功能，為您的第 1 層資料庫、商業智能和高級分析工作負載實現無與倫比的規模、安全性、高可用性和領先性能。
   **標準版 `Standard`**|為中端應用程序獲得豐富的編程功能、安全創新和快速性能。 無需更改任何代碼即可輕鬆升級到企業版。

3. **`SQL Server` 各功能說明**
   >**SQL Server 各功能說明**

   功能簡稱|功能說明
   --|--
   **`SSMS`**|`SQL Server Management Studio` 您可以建立/管理資料庫物件   (如資料庫，表，儲存過程，檢視等)，檢視該資料在資料庫中，組態使用者帳戶，執行備   份，複製，資料庫之間的資料傳輸，等等。
   **`SSDS`**|**`SQL Server Database Engine Services`** `Database    Engine` 的 `SQL Server` 元件是用來儲存、處理及維護資料安全的核心服務。    `Database Engine` 提供控制存取和快速交易處理，以符合企業中最需要的資料耗用   應用程式的需求。`SQL Server` 在單一電腦上最多支援 `50` 個 `Database    Engine` 執行個體。
   **`SSIS`**|**`SQL Server Integration Services`** 用於建立高效能資料   整合方案，包括資料倉儲的擷取、轉換和載入 (`ETL`) 封裝。它包含用於建立及偵錯   封裝的圖形工具和精靈；用於執行工作流程功能 (例如 `FTP` 作業)、執行 `SQL` 陳   述式或傳送電子郵件訊息的工作；用於擷取及載入資料的資料來源和目的地；用於清除、   彙總、合併和複製資料的轉換；用於管理 `Integration Services` 的管理服務，   即 `Integration Services` 服務；以及用於程式設計 `Integration    Services` 物件模型的應用程式發展介面 (`API`)。
   **`SSAS`**|**`SQL Server Analysis Services`** 主要功能是 `BI` 資料   分析解決方案中的一環，提供 `OLAP` 所需要的各項處理邏輯，分析功能，資料建模方   法，從資料模型的設計，分析需求的定義，資料結構的規劃，分析作業的維護與管理，   `SSAS` 都提供相對應的工具、軟體，和服務，`SSAS` 也能夠和其他資料服務軟體彈   性整合，完整又全面的搭建起 `BI` 資料分析解決方案。
   **`SSRS`**|**`SQL Server Reporting Service`** 它是 `SQL Sever` 提   供的報表建立工具。它為 `SQL server` 提供了基於各種資料來源來開發、部署、管   理儀表盤和分頁報表的方式，操作簡單，適用於喜歡在獨立環境中工作而不喜歡在   `Visual Studio/SSDT` 中使用報表設計器的使用者。

## `SQL Server` 應用

1. 企業資料庫
   許多企業選擇 `SQLServer` 作為核心資料庫平台。這個系統能有效管理大量客戶資料、訂單資訊與財務數據。企業常用 `SQLServer` 建置 `ERP`（企業資源規劃）與 `CRM`（客戶關係管理）系統。這些系統需要高效能、高穩定性的資料庫支援。`SQL Server` 支援多伺服器部署，能實現資料備援與高可用性。當主伺服器發生故障時，備援伺服器可自動接手，確保業務不中斷。企業也能利用權限控管功能，保障敏感資料安全。

2. 網站後端
   現代網站需要穩定且高效的資料庫支援。開發團隊常用 `SQL Server` 作為網站後端資料儲存解決方案。這個系統能快速處理大量用戶請求，並支援即時資料查詢。電商平台、會員系統與內容管理系統（`CMS`）都適合採用 `SQL Server`。開發人員可利用 `T-SQL` 語言設計複雜查詢，滿足多元業務需求。 `SQL Server` 也支援雲端部署，讓網站能隨時擴充資源，應對流量高峰。

3. 數據分析
   企業在數據分析領域也大量應用 `SQL Server`。這個系統內建分析服務（`SSAS`），能協助建立多維度資料集。資料分析師可利用 `SQL Server` 進行資料挖掘、趨勢預測與商業智慧（BI）報表製作。企業能將來自不同來源的資料整合至 `SQL Server`，進行統一分析。`SSRS` 報表工具讓決策者能即時取得視覺化分析結果。

4. 工廠自動化
   現代工廠高度依賴數據來提升生產效率。自動化系統需要即時監控設備運作、產線狀態與品質數據。資料庫平台在這些應用中扮演關鍵角色。許多工廠選擇以資料庫管理龐大的感測器數據與生產紀錄。這些資料有助於優化流程、預測維修時機，並確保產品品質。

   資料庫管理系統能協助工廠即時處理大量數據。管理人員可透過直覺式介面快速查詢生產狀況。`SQLExecMS` 等工具支援即時監控伺服器活動，讓工程師能隨時掌握查詢效能與資源使用情形。這些功能有助於發現生產過程中的瓶頸，並提出效能優化建議。工廠也能利用自動化排程功能，定時執行資料備份、設備檢查等維護任務，確保數據即時性與系統穩定。

   資料庫平台還提供進階 `SQL` 編輯功能，協助開發人員設計複雜查詢與報表。這些工具讓工廠能即時追蹤產線異常，並快速做出調整。當工廠導入資料庫管理系統後，生產決策變得更有依據，整體營運效率也隨之提升。

## `SQL Server` 商業智慧(`BI`) 解決方案

1. 商業智慧(`BI`) 解決方案說明
   `SQL Server` 中隸屬於 `Microsoft` 商業智慧平台的功能包括 `Analysis Services`、`Integration Services`、`Master Data Services`、`Reporting Services`，以及數種可以用於建立及使用分析資料的用戶端應用程式。

   `Analysis Services` 及 `Reporting Services` 可以作為獨立伺服器、作為向外擴展的配置，或是作為 `SharePoint` 伺服器陣列中的共用服務應用程式進行安裝。 將這些服務安裝在伺服器陣列中可以啟用僅在 `SharePoint` 中可用的 `BI` 功能，例如 `Power Pivot for SharePoint` 和 `Power View`。 `Power View` 是 `Reporting Services` 提供的新隨選互動式報表設計工具，可以在 `Power Pivot` 或 `Analysis Services` 表格式模型資料庫上執行。

2. 核心功能
   - **資料轉換**
     使用 `SQL Server Analysis Services (SSAS)` 建立多維度或表格式模型，整合來自不同資料來源的資料，建立單一的事實資料集，並利用 `Power BI` 和 `Excel` 等工具進行資料視覺化。
   - **互動式報表**
     `SQL Server Reporting Services (SSRS)` 提供互動式報表功能，讓使用者可以探索資料、尋找模式並做出更快速的決策。`Power BI` 報表伺服器則整合了 `SSRS`，提供符合商業需求的精確報表。
   - **雲端擴展**
     透過 `Power BI`，企業可以連接到內部部署的資料來源，無須將資料移轉到雲端，即可集中檢視所有資訊。 同時，`Power BI` 報表伺服器作為 `SQL Server Enterprise Edition` 的軟體保證利益，讓企業將現有的內部部署報表和資料延伸到雲端。
   - **與 `Power BI` 整合**
     `SQL Server` 的 `BI` 功能與 `Power BI` 緊密整合，可以將報表項目釘選到 `Power BI` 儀表板，實現更豐富的互動體驗。

   整體而言，`SQL Server` 的 `BI` 解決方案，讓企業能夠利用熟悉的工具和技術，將資料轉化為可操作的洞見，進而提升決策品質和效率。
