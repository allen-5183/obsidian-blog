## 安裝 `SSMS`

## 前言

`SQL Server Management Studio （SSMS）` 是管理任何 `SQL` 基礎結構的整合式環境。 使用 `SSMS` 存取、配置、管理、管理及開發使用 `Microsoft SQL` 資料庫引擎的平台，包括 `SQL Server`、 `Azure SQL` 資料庫、 `Microsoft Fabric` 中的 `SQL` 資料庫、 `Azure SQL` 管理實例、 `Azure` 虛擬機上的 `SQL Server` 等。

`SSMS` 提供單一完整的公用程式，結合一組廣泛的圖形工具與許多豐富的腳本編輯器，為所有技能層級的開發人員和資料庫管理員提供 `SQL Server` 的存取權。 欲了解更多資訊，請參閱 [SQL Server Management Studio 中的元件與功能](https://learn.microsoft.com/zh-tw/ssms/components-features)。

## 執行 `SSMS`

1. 執行 `SSMS` 主程式
   ![](https://pic.allen5183.synology.me/SQL_DOWNLOAD_30.png#w60)

2. 輸入登入資訊
   - `Serber Name`: `.\SQL2025`
   - `Authentication`: `SQL Server 驗證`
   - `User Name`: `sa`
   - `Password`: `1qaz@wsx`
   - `Remember Password`: **打勾**
   - `Trust Server Certificate`: **打勾**
   - `Database Name`: `master`
   ![](https://pic.allen5183.synology.me/SQL_DOWNLOAD_31.png#w60)
   ![](https://pic.allen5183.synology.me/SQL_DOWNLOAD_32.png#w60)

   >主機名稱可使用替代方式輸入, 因為 **主機名稱** 及 **IP** 皆有可能變更，因此可以用以下方式來取代
   > - `127.0.0.1`
   > - `localhost`
   > - `.`

3. 查詢 `SQL Server` 資訊
   - 新增查詢
     ![](https://pic.allen5183.synology.me/SQL_DOWNLOAD_33.png#w60)

   - 執行查詢指令
     在查詢視窗中輸入以下指令並按下 **執行** 即可查詢

     ```sql
     SELECT @@VERSION
     ```

     ![](https://pic.allen5183.synology.me/SQL_DOWNLOAD_34.png#w60)

     或是下逹以下指令並按下 **執行** 即可查詢更詳細的資訊

     ```sql
     exec master..xp_msver
     ```

     ![](https://pic.allen5183.synology.me/SQL_DOWNLOAD_35.png#w60)
